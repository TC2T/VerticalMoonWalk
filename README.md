# VerticalMoonWalk

Un petit jeu de plateforme vertical inspiré par l’univers de Michael Jackson, en HTML/CSS/JavaScript.

## Lancer localement

Ouvre simplement [index.html](index.html) dans ton navigateur, ou utilise un serveur local si tu préfères.

## Publier sur GitHub Pages

1. Crée un dépôt GitHub nommé `VerticalMoonWalk`.
2. Dans ton terminal, va dans le dossier du projet puis exécute :

```bash
git init
git branch -M main
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/TON_UTILISATEUR/VerticalMoonWalk.git
git push -u origin main
```

3. Sur GitHub, ouvre les réglages du dépôt, puis Pages.
4. Choisis “Deploy from a branch”, branche `main`, dossier `/ (root)`.
5. Enregistre et attends quelques minutes.

Le lien final sera de la forme :

```text
https://TON_UTILISATEUR.github.io/VerticalMoonWalk/
```

Tu peux ensuite envoyer ce lien à tes proches.
